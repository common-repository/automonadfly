Automatic Shorten a URL in content post using AdFly and masked with other Shortner Services

== Description ==

AutoMonAdFly is a wordpress plugins that will automatically parse and shorter your external URL in Post Content. Not only that, your adfly url can also masked using other shortener url service so then your visitor will see there is no adfly link on your site.

** If you want cache works, you need this plugins [WP Plugin Cache](http://wordpress.org/extend/plugins/wp-plugin-cache/ "WP Plugin Cache") **

** Need a CURL, how to check it? this is the tutorial [How to know the curl ready on the server](http://inhs.web.id/2010/11/is-your-curl-ready/ "How to know the curl ready on the server") **
